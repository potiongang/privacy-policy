# Privacy Policy

**Last updated:** December 17, 2025

## 1. Introduction

PotionGang.fr is a browser extension designed to assist users by automating certain repetitive interactions encountered during web navigation.

Protecting user privacy is a priority. This policy explains clearly what data is (and is not) collected.

---

## 2. Data Collection

**PotionGang.fr does not collect any personal data.**

Specifically, it does **not** collect:

* personal identification information
* browsing history
* page content
* user input data
* IP addresses
* cookies
* analytics or tracking data

---

## 3. Data Usage

Since no data is collected, **no data is used, stored, or processed**.

## 4. Data Sharing

PotionGang.fr:

* does not share any data with third parties
* does not transmit any information to external servers
* does not use analytics or tracking services

---

## 5. Permissions

The permissions requested by the extension are used solely to enable its core functionality:

* **tabs**: required to manage tabs opened during certain navigation steps
* **limited website access**: the extension only operates on pages necessary for its functionality

No permissions are used for data collection purposes.

---

## 6. Security

PotionGang.fr operates without any backend server and does not transmit data outside the browser, reducing security and privacy risks.
